# 学途技能 · 微信小程序 MVP

这是一个面向德阳本地的大学生非学科类技能分享撮合小程序。项目已经从静态 HTML 原型改为微信原生小程序 + 云开发结构。

## 当前范围

第一版只做真实撮合闭环，不接支付：

- 微信云开发登录
- 学习方发布非学科类技能需求
- 达人完善资料并上传在读证明
- 达人认证通过后申请接单
- 匹配后生成会话
- 云数据库持久化
- 云函数统一处理写入和权限校验

不包含：

- 微信支付
- 订单结算与资金监管
- 管理后台 UI
- 自动审核
- 真实客服和纠纷仲裁

## 目录结构

```text
.
├── miniprogram/              # 微信小程序原生页面
│   ├── pages/                # 广场、发布、消息、我的、详情、资料编辑
│   ├── shared/               # 前端可复用领域规则和常量
│   └── utils/api.js          # 小程序端云函数调用封装
├── cloudfunctions/api/       # 统一云函数入口
├── database/                 # 云数据库 schema 和种子数据
├── tests/                    # Node 领域规则测试
└── project.config.json       # 微信开发者工具项目配置
```

## 本地验证

```bash
node --test tests/domain.test.js
```

## 微信开发者工具运行

1. 用微信开发者工具打开项目根目录。
2. 把 `project.config.json` 里的 `appid` 改成你自己的小程序 AppID。
3. 开通云开发环境。
4. 在开发者工具里把云函数根目录识别为 `cloudfunctions/`。
5. 上传并部署 `cloudfunctions/api`，安装依赖。
6. 按 `database/schema.md` 创建集合和索引。
7. 可用 `database/seed.json` 手工导入初始达人和需求数据。

## 关键工程规则

- 不再使用 `innerHTML` 或浏览器 DOM。
- 页面只通过 WXML 数据绑定渲染用户输入。
- 所有写操作必须走 `api` 云函数。
- 达人未通过认证不能申请需求。
- 技能范围固定为非学科类方向，避免把项目做成中小学学科培训平台。
