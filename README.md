# 学途技能 · 大学生技能分享平台原型

大学生技能分享 × 兴趣学习陪伴的撮合平台原型（纯前端静态页面，无后端）。

⚠️ **定位说明**：本产品仅覆盖编程入门、绘画设计、乐器声乐、摄影摄像、口语交流、运动技能、书法棋艺、剪辑运营等**非学科类**兴趣技能方向，不涉及语文/数学/英语等学科类课外培训内容，目前仅限德阳市范围。

## 目录结构

```
.
├── index.html    # 页面结构
├── style.css     # 样式（CSS 变量做主题色管理）
├── app.js        # 交互逻辑与模拟数据（无真实后端，数据保存在内存中，刷新会重置）
└── README.md
```

## 本地预览

无需安装任何依赖，直接用浏览器打开 `index.html` 即可，或者用任意静态服务器：

```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 功能概览

- 双角色：学习需求方 / 技能达人，可在「我的」页面一键切换（演示用）
- 技能广场：按技能类别筛选，浏览达人卡片或需求卡片
- 发布需求 / 申请接单：模拟撮合确认流程（1.5秒后自动模拟对方确认）
- 消息中心：系统通知 + 匹配成功后的模拟聊天
- 资料与认证：达人可编辑资料、上传认证材料（模拟审核，2秒后自动通过）
- 收费标准：线上/线下服务费比例不同（当前设置：线上 10%，线下 15%），在达人详情页展示课时费与到手金额对照

## 说明

这是产品原型/演示项目，所有数据均为前端模拟数据，不含真实后端、支付、鉴权逻辑。正式开发前建议：

1. 就经营范围、平台备案要求向当地主管部门或专业律师做合规确认
2. 补充真实的用户系统、支付与资金监管方案、审核后台
3. 如需接入微信小程序原生框架（WXML/WXSS/JS + 云开发），需另行改写页面结构
